about & credits

mapping music

about

music is tied to our cherished spaces and memories. whether a song reminds you of home, your favorite person, the worst time in your life or the best, music brings us to a time and place. on “mapping music”, you can share those memories, as well as explore others’ nearby (and across the world).


“mapping music” was made as my final project in “intro to web design” at NYU. It was inspired by queering the map, a site where users can pin their queer memories on a map.


code

The map data is from OpenStreetMap and presented by Leaflet, which I learned how to use in a prior JavaScript class, but solidified my code using their quick start guide. I’m using Google Firebase to store the pin data, and learned how to implement that into my map here. All credit is properly listed within the JavaScript file as well.


All credit is given in my JavaScript and CSS files. All other HTML, CSS and JavaScript was written by me, including text, images, semantic elements, grids & flexboxes, menu bar styling and DOM queries, functions, and @media queries.
